The downloadTotal read-only property of the {{domxref("BackgroundFetchRegistration")}} interface returns the total amount of data to be downloaded. This is the value provided in the options object passed to {{domxref("BackgroundFetchManager.fetch()")}}.

Syntax

var downloadTotal = BackgroundFetchRegistration.downloadTotal;

Value

A {{jsxref("Number")}}.
