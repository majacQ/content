The BackgroundFetchRegistration interface of the {{domxref('Background Fetch API','','',' ')}} provides information about an initiated background fetch. A new instance is created whenever {{domxref("BackgroundFetchManager.fetch()")}} is called with a developer defined ID and stored by the {{domxxref("BackgroundFetchManager")}}.
